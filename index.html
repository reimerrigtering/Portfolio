Design

Meet and Play

I designed a social gathering space for teenagers to meet on sunday evenings so they can meet up and play games.

Art for Curiosity

This artwork was made in a team with the goal to stimulate senses. Using sound, lights and haptics to pull the user in and let the surroundings fade away.

Display Creativity

Over time, I equipped myself with multiple different tools to show instead of telling. Using basic skeching on paper, skethcing digitally or 3D modeling, all of them create a "show, don't tell"-environment for a team of people to work in.

Coding

Speaking Computers

The Quantum Poiesis project is a project where I did the coding for an artist for this piece. It is the first step towards a quantum computer speaking old English.

UI is Fun

I coded many small personal projects using python, many times making easy small games using Pygame. Wanting an easier way to create UI elements I created my own library to support Pygame and make UI elements trivial. Using this, creating scenes, animations, buttons, bars and so much more is only a few lines of code while maintaining a great amount of customizability.

Attention Gamers

Two times I created games in Unity with a group of people. Using C# to create relative complex games within a matter of weeks. Coding different mechanics for the player to enjoy while playing the game.
